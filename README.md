# 📊 Data Science Mini Projects Collection

This repository contains three beginner-friendly Data Science projects designed to build a strong foundation in data analysis, visualization, and machine learning using Python.

---

## 📁 Projects Overview

### 🔹 Project 1: Exploratory Data Analysis (EDA)

Perform detailed data analysis on a publicly available dataset such as the **Iris**, **Titanic**, or any dataset from Kaggle.

#### 📌 Objectives:

* Load and explore the dataset
* Handle missing values
* Perform statistical analysis
* Identify patterns and relationships

#### 🛠️ Tools & Libraries:

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

#### 📊 Key Tasks:

* Data cleaning and preprocessing
* Summary statistics (mean, median, std, etc.)
* Data visualization:

  * Histograms
  * Box plots
  * Heatmaps (correlation matrix)
  * Pair plots

#### 📈 Outcome:

Understanding of dataset structure, trends, correlations, and key insights.

---

### 🔹 Project 2: Simple Linear Regression on Housing Prices

Build a regression model to predict housing prices using features like number of rooms, area, etc.

#### 📌 Objectives:

* Perform feature selection
* Normalize/scale data
* Train a linear regression model
* Evaluate model performance

#### 🛠️ Tools & Libraries:

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

#### 📊 Key Tasks:

* Data preprocessing
* Splitting dataset into training & testing sets
* Model training using Linear Regression
* Model evaluation using:

  * Mean Squared Error (MSE)
  * R² Score

#### 📈 Outcome:

A predictive model capable of estimating house prices based on input features.

---

### 🔹 Project 3: Loan Eligibility Prediction

Develop a machine learning model to predict whether a loan application will be approved or not.

#### 📌 Objectives:

* Understand dataset structure
* Perform data cleaning
* Encode categorical variables
* Train classification models

#### 🛠️ Tools & Libraries:

* Python
* Pandas
* NumPy
* Scikit-learn
* Seaborn

#### 📊 Key Tasks:

* Handling missing values
* Label encoding / one-hot encoding
* Feature selection
* Model training using algorithms like:

  * Logistic Regression
  * Decision Tree
  * Random Forest
* Model evaluation using:

  * Accuracy
  * Confusion Matrix
  * Classification Report

#### 📈 Outcome:

A classification model that predicts loan approval status based on applicant data.

---

## 📂 Dataset Information

### 📌 Project 1 Dataset

* Public dataset (Iris / Titanic / Kaggle dataset)
* Contains numerical and categorical features
* Used for analysis and visualization

### 📌 Project 2 Dataset

* Housing dataset (e.g., Boston Housing dataset)
* Features include:

  * Number of rooms
  * Area
  * Location factors
* Target variable: House Price

### 📌 Project 3 Dataset

* Loan dataset with features such as:

  * Gender
  * Marital Status
  * Income
  * Loan Amount
  * Credit History
* Target variable:

  * Loan Status (Approved / Not Approved)

## 🚀 How to Run the Projects

1. Clone the repository:

```bash
git clone https://github.com/creativework928-max/HexSoftwres_Project-01
```

2. Navigate to the project folder:

```bash
cd data-science-projects
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Open Jupyter Notebook:

```bash
jupyter notebook
```
